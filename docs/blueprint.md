# **App Name**: Xmarty Creator

## Core Features:

- Themed Layout: A themed, structured layout featuring a navigation bar, main content area, and a footer link and footer.
- Welcome Section: A welcome section with educational cards and a carousel.
- Theming: Enable switching between light and dark themes for better user experience based on user preferences.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) evoking a sense of calm, learning, and openness.
- Background color: Very light blue (#F0F8FF), almost white, providing a clean and unobtrusive backdrop.
- Accent color: Soft yellow (#FAFAD2) to highlight important elements, complementing the blue tones with a touch of warmth.
- Body font: 'PT Sans', sans-serif.
- Headline font: 'Playfair', modern serif. Since Playfair is only suitable for short amounts of text, PT Sans is also used for body text
- Use simple, outlined icons that relate to education topics. These should maintain a consistent style across the platform to enhance the clean and modern look.
- The layout should be responsive and adapt to different screen sizes. Utilize a grid system for consistent spacing and alignment of elements.
- Incorporate subtle animations for transitions and feedback, making the interaction more engaging and intuitive.
